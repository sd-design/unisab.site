Комплексная автоматизация банковской деятельности с 1995 года. Основной продукт компании АБС «UBS.NET» – полнофункциональное многокомпонентное решение для автоматизации деятельности банков.

Последние новости

  • Новые возможности
    системы ДБО UBS.HOME
    в части авторизации, идентификации и регистрации новых клиентов с использованием ЕСИА/ЕБС – взаимодействие системы ДБО с ЕСИА/ЕБС через интеграционную платформу РНД СОФТ
    29.12.2022

    Интеграционное решение существенно снижает для банков ресурсоемкость создания и обслуживания программно-аппаратной инфраструктуры для взаимодействия с ЕСИА/ЕБС, обеспечивая требуемый уровень информационной безопасности.




  • Для взаимодействия с платформой личных финансов Финуслуги Московской биржи по сберегательным продуктам (вкладам) создан и успешно внедрен
    в ББР Банк (АО) новый бизнес-модуль АБС «UBS.NET» «Взаимодействие с MOEX (ДЕПОЗИТЫ)»
    30.12.2022

    Новая функциональность:
    - предоставляет возможность управления продуктовым каталогом, публикуемым на витринах Платформы;
    - автоматизирует взаимодействие с Платформой на всех этапах жизненного цикла договоров вклада;
    - обеспечивает контроль взаимодействия через систему мониторинга, отражающую потоки взаимодействия с Платформой.

История компании

Преимущества UBS.NET

  • On-line система с возможностью доступа клиентов 24 часа в сутки и быстрая реакция технической поддержки по обращениям

  • Интеграция с системами электронных платежей, денежных переводов и карточными процессингами.

  • Автоматизация всего спектра банковских услуг для бизнеса любого масштаба с возможностью выполнения проектов «под ключ»

  • Интеграция с государственными информационными системами (ГИС ГМП, ГИС ЖКХ, ФССП, ФНС, ФМС, ФТС, ПФР)

  • Система дистанционного банковского обслуживания для клиентов банка (Web-интерфейс и мобильное приложение)

  • Интеграция с системой быстрых платежей СБП (C2C, С2B, B2C, СБПэй, мобильное приложение ТСП)

  • Полный спектр банковской отчетности, интеграция с БКИ, МИ ФНС, ЕПВВ ЦБ России

  • Универсальная единая технологическая платформа для большинства современных операционных систем и сред исполнения

  • Повышенная устойчивость и безопасность приложений, включая исполнение кода сторонних производителей
